# Assets Directory

Поместите сюда медиа-файлы для бота:

## Необходимые файлы:

1. **guide.pdf** - PDF-гайд "Как не забывать слова и говорить свободно"
   - Этот файл будет отправляться пользователю при нажатии кнопки "Получить гайд"

2. **audio_task3.mp3** - Аудио для задания 3
   - Должно содержать фразу: "I sit, drink coffee, and watch people walking outside."
   - Формат: MP3

## Инструкция:

1. Создайте или получите необходимые файлы
2. Поместите их в эту директорию с указанными именами
3. Бот автоматически обнаружит и будет использовать эти файлы

**Примечание:** Если файлы отсутствуют, бот будет работать с текстовыми заглушками.
