# Dimabot

Telegram бот, который отправляет гайд и приглашает пройти мини-урок.

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Экспортируйте токен бота:
   ```bash
   export BOT_TOKEN="ваш_telegram_бот_токен"
   ```
   Необязательные переменные:
   - `GUIDE_PATH` — путь к PDF-файлу, если вы хотите использовать другой документ.
   - `MINI_LESSON_URL` — ссылка на мини-урок (если указано, кнопка откроет ссылку вместо callback).
3. Запустите бота:
   ```bash
   python -m dimabot.main
   ```

PDF-файл по умолчанию расположен в каталоге `guide_assets/guide.pdf`.
